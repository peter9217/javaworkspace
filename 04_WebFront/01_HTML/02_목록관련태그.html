ol(ordered list) 태그 : 순서가 있는 목록

        type 속성 : 순서를 나타내는 형식을 지정
        - 미작성 : 1 부터 1씩 증가하는 숫자
        - type="a" :영어 소문자 순서 (abcd...)
        - type="A" :영어 소문자 순서 (abcd...)
        - type="i" :로마  소문자 순서 
        - type="I" :로마  대문자 순서 

        start 속성 : 몇 부터 시작?
        - start="5" -> 5부터 시작
        reversed : 반대
  1. 아아
  2. 삼김
  3. 고
  4. 히
  5. ㅁㄴㅇ
